# МОЯ ИНСТРУКЦИЯ (my_manual)

## Запуск локально
```
npm install
npm run dev
```
Сайт: `http://localhost:4321/`
Админка: `http://localhost:4321/admin/`

## Как писать заметки (вручную)
Файлы: `src/content/notes/*.md`

Шаблон:
```
---
title: Заголовок
structural: ЗАМЕТКИ | ДИАЛОГИ | ИНСТРУКЦИИ | НА ЗАМЕТКУ
topics:
  - ЛИНУКС
---
Текст заметки (1–2 строки).
```

## Как писать заметки через админку (Decap)
1) Открой `/admin/`.
2) Авторизуйся через GitHub.
3) Создай/открой заметку.
4) Укажи:
   - Title
   - Structural (тип)
   - Topics (одна тема)
5) Сохрани → коммит в репозиторий.

## Как добавлять схемы
1) Положи картинку в: `public/img/schemes/`.
2) Создай файл в: `src/content/schemes/`.

Шаблон:
```
---
title: Схема VSC
topics:
  - VSC
image: /img/schemes/vsc.png
---
```

## Основные правила
- `topics` всегда массив (даже если 1 тема).
- `date` нигде не используется и не нужен.
- Схемы показываются в разделе «СХЕМЫ» и в «ЗАМЕТКИ».
- Для схем есть отдельная страница просмотра `/schemes/[slug]`.
